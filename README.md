## The Problem

India's SHG ecosystem is the largest grassroots women's network in the world. Yet most members still lack the practical skills to earn sustainably.

| | |
|---|---|
| Women mobilized | **10.05 Crore** |
| Active SHGs | **90.90 Lakh** |
| Districts covered | **700+** |
| With real enterprise skills | **far too few** |

Current training is static, trainer-dependent, built for individuals вҖ” and completely disconnected from real income generation.

---

## The Solution

Udyam Guild turns each SHG into a **guild** and each lesson into a **business mission.** Women don't watch lectures. They complete real tasks, earn XP, climb a live leaderboard against other guilds in their cluster, and unlock actual market access as they level up.

## The 5 Pillars

**Pillar 1 вҖ” Guild Identity & Onboarding**
Each SHG joins as a named guild with 8вҖ“12 members. Members pick one of 5 roles: Treasurer, Catalog Lead, Seller, Maker, or Digital Champion. A Baseline Assessment generates a starting Loan Readiness Score. KRP/NGO staff join as Guild Guides with cluster-level oversight tools.

**Pillar 2 вҖ” Structured Learning Journeys**
7вҖ“10 week tracks mapped to real business goals вҖ” "Start Selling Locally", "Digital Orders", "Festival Business". Each lesson is a 3-minute audio in Hindi or a regional language, followed by a scenario simulation. Content is role-filtered: a Treasurer gets pricing content, a Seller gets negotiation content.

**Pillar 3 вҖ” Mission Engine**
6 levels from Udaan to Udyam Pro. Each level unlocks only after completing real-world tasks вҖ” not just watching videos.

```
Level 1  Udaan         Digital basics, product idea, confidence building
Level 2  Tayyari       Pricing, cost understanding, customer communication
Level 3  Bazaar Ready  в”Җв”Җв”Җв”Җ Meesho listing UNLOCKED
Level 4  Vishwas       Payments, record keeping, repeat buyers
Level 5  Pragati       в”Җв”Җв”Җв”Җ ONDC + e-NAM marketplace UNLOCKED
Level 6  Udyam Pro     Advanced growth, facilitator certification
```

Daily Sparks, Weekly Quests, and Cluster Leaderboards keep the engagement loop running.

**Pillar 4 вҖ” Business Execution Tools**

- Smart Pricing Calculator вҖ” cost + margin gives the correct selling price
- Voice Cashbook вҖ” record daily income by speaking
- Product Catalog Builder вҖ” photos and descriptions become a shareable catalog
- Income Logger вҖ” monthly tracking with streak badges
- Loan Readiness Score вҖ” 0 to 100 score, exportable directly to MFI partners

**Pillar 5 вҖ” Facilitator Intelligence + Market Linkage**

- Guild Heatmap вҖ” facilitators see at a glance which guilds need attention
- WhatsApp Nudge Engine вҖ” automated mission reminders sent to members
- Market Linkage Hub вҖ” Meesho to ONDC to e-NAM, progression-gated
- MFI Integration вҖ” automated credit proxy scores pushed to partner banks

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL 16
- Redis 7
- Docker + Docker Compose (recommended for local dev)

### Quick start with Docker

```bash
git clone https://github.com/teentitans/udyam-guild.git
cd udyam-guild

cp .env.example .env
# fill in your values вҖ” see Environment Variables below

docker-compose up --build
# Frontend вҶ’ http://localhost:5173
# API      вҶ’ http://localhost:5000
```

### Manual setup

```bash
# Root
npm install

# Frontend
cd client && npm install && npm run dev

# Backend
cd ../server && npm install && npm run dev

# Database вҖ” run once after Postgres is up
psql -U postgres -f database/schema.sql
psql -U postgres -f database/seeds/01_missions_level1.sql
psql -U postgres -f database/seeds/03_demo_guilds.sql
```

### Environment variables

Copy `.env.example` to `.env` and fill in:

```
PORT=5000
NODE_ENV=development

DATABASE_URL=postgresql://postgres:password@localhost:5432/udyam_guild
REDIS_URL=redis://localhost:6379

JWT_SECRET=change_this_to_a_long_random_string
JWT_EXPIRES_IN=7d

# set to 'mock' during development вҖ” OTP will always be 1234
OTP_MODE=mock
FAST2SMS_API_KEY=

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_BUCKET_NAME=udyam-guild-assets
AWS_REGION=ap-south-1

TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_WHATSAPP_FROM=whatsapp:+14155238886

MEESHO_PARTNER_API_KEY=
MEESHO_API_BASE_URL=https://api.meesho.com/v1

VITE_API_BASE_URL=http://localhost:5000/api
```
